Features

Everything you need.
Nothing you don't.

Lightweight, zero-latency, and built to run quietly in the background all day.

🎹
8 Built-in Profiles

From deep NK Cream thock to loud Box Navy clatter — each profile is a meticulously crafted synth voice.

🌐
Global Keyboard Hook

OS-level capture via uiohook-napi. Contsound hears every keypress even when a game has focus.

📂
Custom Audio Import

Drag any .wav, .mp3, .ogg, or .m4a file onto the app to use your own sound. It saves automatically.

⚡
Zero Latency

Web Audio API synthesis. No perceptible delay — the sound hits the exact moment you press.

🔇
No Key Repeat Spam

Holding a key fires the sound once. Smart tracking means no stutter, no machine-gun clicking.

🗂️
Tray-Based

Minimises to the system tray, always listening, never in the way. Close the window — it keeps running.

How It Works

Up and running in 30 seconds.

01
Download & Run

No installer. Download the .exe and double-click — Contsound appears in your system tray instantly.

02
Pick a Profile

Choose from 8 built-in synth voices or drag in your own audio file for a fully custom sound.

03
Type Normally

Every keypress — browser, IDE, game — now carries the satisfying click of a real mechanical switch.

04
Tune to Taste

Open Advanced Controls to dial in volume, pitch, and decay until it sounds exactly right.
